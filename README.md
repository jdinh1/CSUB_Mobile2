# CSUB Mobile
A Android mobile application designed for CSUB as part of a senior project. Its purpose is to provide students and visitors with information about the university.

## Features
1. Campus Map
2. News
3. Events
4. Directory
5. Social Media
6. Dining Locations
7. Transportation
8. Schedule
9. Moodle/BlackBoard
